MatLab ортасында жалғыз импульстерді генерациялау үшін кірістірілген функциялар бар:

Tripuls функциясы жалғыз амплитудасы бар үшбұрышты импульсті қалыптастырады. Осы функцияны шақыру пішімі келесі:

t - уақыт мәндерінің векторы, width - импульстің ені (ұзақтығы), skew - импульстің жоғарғы қалпын анықтайтын ассимметрия коэффициенті. Импульс шыңы t = width*skew/2 кезінде орналасқан. Skew параметрі 1-ден 1-ге дейінгі ауқымда жатуы керек.

Теориялық мәліметтер

Сигналдардың күрделі тізбектерін қалыптастыруды сызықты математикалық операциялар – константаға көбейту, қосу және уақытша жылжу арқылы жүзеге асыруға болады. Мысал ретінде тиісті гармоникалық құрамдастардан тікбұрышты импульстер тізбегінің қалыптасуын қарастырайық.

Импульстік периодтық сигналдың гармоникалық құрауыштарға ыдырауын сипаттайтын формула:



Үлгі № 1

Сегіз гармоникалық құрамдас бөліктер үшін осы функцияны іске асыратын бағдарлама келесідей):

N=8; % нөлдік емес гармониктер саны
t = -1:0.01: 1; % уақыт моменттерінің векторы
A = 1; % амплитудасы
T = 1; % кезеңі
nh = (1:n)*2-1; нөлдік емес гармоникалардың нөмірі %
% жолдарды есептеу-гармоника
harmonics = cos(2 * pi*nh' * t / T);
Am = 2 / pi./nh; % гармоника амплитудасы
Am (2:2:end)=-Am(2:2:end); % таңбалардың ауысуы
s1 = harmonics.* repmat(Am', 1, length(t));
% гармониканың жеке қосынды қалыптастыру
s2 = cumsum(s1);
for k=1:N,
subplot (4,2,k); plot (t, s2(k,:))
end

- Square, sawtooth, diric және chirp мүмкіндіктерін өзіңіз зерттеңіз. Ол үшін S – зерттелетін сигнал, [] – оның мәндерінің диапазоны, Fs – дискретизация жиілігі).
- Randn (m, n) функциясының көмегімен жасалатын кездейсоқ сигнал үшін М-файлды жасаңыз, онда: сигнал пайда болатын, ол үшін орташа мән есептелген, СКО және гистограмма салынған.
- Кездейсоқ шу қосылған М-файлын жасаңыз. Мұндай сигналдың статистикалық сипаттамаларын есептеңіз.
- М-файлын жасаңыз, онда гармоникалық сигналға берілген қатынаста ақ шу қосылады (бұл үшін awgn функциясын пайдаланыңыз).
- 16 санаудан тұратын сигналды генерациялаңыз. Бұл сигнал үшін М-файлын жасаңыз, ол келесі сигналды ыдырату нұсқаларын іске асырады:

1. импульстік ыдырау;
2. сатылы ыдырау;
3. жұп/тақ ыдырау;
4. ауыспалы ыдырау.

№5 Тәжірбиелік сабақ



Сигналдардың оралуы. Оралу қасиеттері

Тәжіpибелік сабақтың мақсаты: орау операциясын пайдалана отырып, сигналдарды өңдеуді үйрену.

Теориялық мәліметтер

Кіріс сигналы бар сызықтық жүйенің өзара әрекеттесуі оралудың математикалық операциясы арқылы сипатталады:

X[n] * h[n] = y[n], мұнда
X[n] – кіріс сигналы, h[n] – сызықтық жүйенің импульстік сипаттамасы, y[n] - шығыс сигналы. Сызықтық жүйенің қасиеттері оның импульстік сипаттамасымен толық анықталады.

Кез келген жүйені тексеру оған тест сигналын беру әдісімен жүзеге асырылады, кейін осы сигналға реакцияны талдау жүргізіледі.

Жұмыcтың оpындaлу тәpтібі:

MatLab ортасында төмен жиілікті (баяу өзгеретін үшбұрышты импульс) және жоғары жиілікті (синусоидалды сигнал) құрамдастары бар тест сигналын жасаңыз. Тест сигналы 81 есептеуден тұрады. Мұндай сигналды қалыптастыратын бағдарламаның мысалы төменде көрсетілген:

a = zeros(1,81);
c = zeros(1,81);
for i = 10:70
a(i)=a(i-1)+ 0.05;
end
t=0:pi/10:6*pi;
b=sin(t);
for i = 10:70
c(i)=a(i)- b(i-9);
end
plot(c);
grid on
