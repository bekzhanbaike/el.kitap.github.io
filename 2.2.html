2.2. М-файлдар: сценарийлер мен функциялар



MatLab командаларын орындаудың ең қолайлы тәсілі - М-файлдарды қолдану, онда командаларды таңдауға, оларды бірден немесе бөліктермен орындауға, файлда сақтауға және одан әрі пайдалануға болады. М-файлдармен жұмыс істеу үшін М-файлдардың редакторы бар. Бұл редактордың көмегімен өз функцияларын жасап, оларды, соның ішінде командалық жолдан шақыруға болады.

MatLab негізгі терезесінің File мәзірін ашып, New тармағында M-file тармағын таңдаңыз. Жаңа файл М-файлдар редакторының терезесінде ашылады [7].

Бір графикалық терезеде екі графикті құрастыруға әкелетін командаларды редактордан теріңіз:

x = [0:0.1:7];
f = exp (- x);
subplot(1, 2, 1)
plot(x, f)
g = sin (x);
subplot(1, 2, 2)
plot(x, g)

MatLab каталогы work ішкі каталогында, Save as мәзірі file редакторын таңдау арқылы mydemo атымен файлды сақтау қажет. Файлдағы барлық командаларды орындау үшін Debug мәзіріндегі Run тармағын таңдау керек. Экранда Figure No графикалық терезесі пайда болады. Егер сіз синустың орнына косинус графигін салуды шешсеңіз, G = sin(x) жолын G = cos(x) М-файлда өзгертіп, барлық командаларды қайтадан іске қосыңыз.

М-файл түрлері

MatLab-да М-файлдар екі түрден тұрады: командалар тізбегі бар файл-бағдарламалар (Script M-Files), және файл-функциялар (Function M-Files), онда пайдаланушы анықтайтын функциялар сипатталатын болады.

Файл-бағдарламаны (файл-процедураны) жоғарыда қарастырған болатынбыз. Файл-бағдарламада жарияланған барлық айнымалылар орындалғаннан кейін жұмыс ортасында қол жетімді болады. М редакторында жұмыс ортасының мазмұнын көру үшін командалық жолда whos командасын теріңіз.

Командалық терезеде айнымалылардың сипаттамасы пайда болады:

"whos
Name Size Bytes Class
f 1x71 568 double array
G 1x71 568 double array
x 1x71 568 double array
Grand total is 213 elements using 1704 bytes


Бір файл-бағдарламада анықталған айнымалыларды басқа файл-бағдарламаларда және командалық жолда орындалатын командаларда пайдалануға болады. Файл-бағдарламада қамтылған командаларды орындау екі тәсілмен жүзеге асырылады:

- жоғарыда сипатталғандай М-файл редакторынан;
- командалық жолынан немесе басқа файл-бағдарламалардан команда ретінде М-файлдың аты қолданылады.

Қолданудың екінші әдісі әлдеқайда ыңғайлы, әсіресе, егер құрылған файл-бағдарлама бірнеше рет пайдаланылуы мүмкін болған жағдайда. Шын мәнінде, құрылған М-файл MatLab түсінетін команда болады. Барлық графикалық терезелерді жабыңыз және my demo командалық жолында теріңіз, mydemo файл-бағдарлама командаларына сәйкес келетін графикалық терезе пайда болады. m.Mydemo MatLab командасын енгізгеннен кейін келесі әрекеттерді жасайды.

Енгізілген команда жұмыс ортасында анықталған қандай да бір айнымалылардың аты болып табылатындығын тексереді. Егер айнымалы енгізілсе, оның мәні шығарылады.

Егер айнымалы болмаса, онда MatLab енгізілген команданы кірістірілген функциялардың ішінен іздейді. Егер команда кірістірілген функциямен көрсетілсе, онда ол орындалады.

Егер айнымалы және кірістірілген емес функция енгізілмесе, онда MatLab команда атауы және M кеңейтілуі бар М-файлды іздеуді бастайды. Іздеу ағымдағы каталогтан (Current Directory) басталады, Егер М-файлда табылмаса, онда MatLab іздеу жолында орнатылған каталогтарды (Path) қарайды. Табылған М-файл MatLab-да орындалады [7].

Егер жоғарыда аталған әрекеттердің ешқайсысы табысқа әкелмесе, онда командалық терезеге хабарлама шығарылады, мысалы:

"mydem
??? Undefined function or variable 'mydem'.

Әдетте, файлдар пайдаланушы каталогында сақталады. MatLab ортасы оларды табуы үшін, М-файлдардың орналасуын көрсететін жолдарды орнату керек.

М-файл құру

M-файлдары мәтіндік редактордың көмегімен жасалатын қарапайым мәтіндік файлдар болып табылады. Дербес компьютердің операциялық жүйесі үшін MatLab ортасы арнайы кірістірілген редакторды қолданады, дегенмен ASCII-кодымен кез келген басқа мәтіндік редактордыда қолдануға болады.

Редакторды екі жолмен ашуға болады:

- File мәзірінен New, содан кейін M-File параметрін таңдау арқылы;
- Еdit өңдеу командасын пайдалану.

M-функциялар кіріс және шығыс аргументтерінің болуына жол беретін M-файлдары болып табылады. Олар MatLab ортасының жұмыс аймағынан бөлек өз жұмыс облысы аймағында ауыспалы жұмыс істейді.

Мысалы:

Average функциясы-Вектор элементтерінің орташа мәнін есептейтін қарапайым M-файл:

function y = average (x)
% AVERAGE Вектор элементтерінің орташа мәні.
% AVERAGE (X), мұнда X – вектор. Элементтердің орташа мәнін есептейді
% Вектор.
% Егер кіріс аргументі Вектор болмаса, қате жасалады.
[m, n] = size(x);
if (~(((m = = 1) | (n = 1)) | (m = = 1 & n = = 1))
error ('кіріс алабы Вектор болуы тиіс')
end
y =sum(x) / length(x); % нақты есептеу

Бұл командаларды average деп аталатын M-файлға енгізіп көріңіз. Average функциясы жалғыз шығу және жалғыз шығу аргументтерін береді. Average функциясын шақыру үшін келесі операторларды енгізу керек:

z = 1:99;
average(z)
